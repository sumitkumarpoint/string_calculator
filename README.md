# String Calculator (Ruby)

## Overview

This project is an implementation of the String Calculator Kata using **Test-Driven Development (TDD)** in Ruby.

The goal is to demonstrate clean code practices, incremental development, and strong testing discipline.

---

## Features Implemented

* Handle empty string → returns 0
* Handle single number
* Handle multiple numbers
* Support new line delimiters (`\n`)
* Support custom delimiters (`//;\n`)
* Handle negative numbers (throws exception)
* Support multiple delimiters (`//[*][%]\n`)
* Support delimiters of any length (`//[***]\n`)

---

## Tech Stack

* Language: Ruby
* Testing Framework: RSpec

---

## How to Run

### Install dependencies

```bash
gem install rspec
```

### Run tests

```bash
rspec
```
